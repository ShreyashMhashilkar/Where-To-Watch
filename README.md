# Where-To-Watch
Where To Watch helps to find OTT platform of the series or movies the user need to watch. User can input the name of movie or series and will get the result with OTT platform name.

It is developed in python and flask. The python library used to extract data is BeautifulSoup.

Steps to follow:

1)Install flask using the command

```

pip install flask

```

2)Install BeautifulSoup using the command

```

pip install bs4

```

3)Run the below command to start the project

```

py app.py

```
